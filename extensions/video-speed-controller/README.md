# Video Speed Controller Extension

Bu eklenti, herhangi bir web sayfasındaki HTML5 videolarının hızını klavye kısayolları ile kontrol etmenizi sağlar.

## Özellikler
- **D tuşu:** Hızı 0.10 artırır.
- **S tuşu:** Hızı 0.10 azaltır.
- Ekranın sağ üst köşesinde güncel hızı gösteren şık bir gösterge paneli.

## Kurulum
1. Chrome tarayıcınızda `chrome://extensions/` adresine gidin.
2. Sağ üst köşedeki **Geliştirici modu (Developer mode)** seçeneğini aktif hale getirin.
3. **Paketlenmemiş öğe yükle (Load unpacked)** butonuna tıklayın.
4. Bu klasörü (`extensions/video-speed-controller`) seçin.

## Kullanım
Herhangi bir video içeren sayfada (YouTube, Netflix, vb.) `D` veya `S` tuşlarına basarak hızı değiştirebilirsiniz.
